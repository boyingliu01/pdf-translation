# 黄道吉日查询工具 — 设计文档

**日期：** 2026-03-07
**状态：** 已批准，待实现
**场景：** 修坟/动土吉日查询，适老化网页工具

---

## 背景与目标

用户父母需在清明节前选择适合修缮祖坟的黄道吉日。工具需结合传统农历黄历规则与用户个人生辰信息（生肖冲煞），给出直观的吉日推荐。

设计为通用工具，支持查询任意月份，未来可供其他用户使用。

---

## 技术栈

- **框架：** Vue 3 + Vite
- **农历/黄历库：** `lunar-javascript`（npm，基于传统典籍，宜忌/黄道黑道数据可靠）
- **样式：** 纯 CSS 自定义（大字体适老化，不引入重型 UI 框架）
- **部署：** 纯静态文件，无后端，可部署到任意静态托管

## 吉日计算逻辑

基于 `lunar-javascript` 提供的数据，综合三个维度评分：

### 维度一：黄历宜忌
- 当天「宜」列表包含「动土」「修造」「破土」「安葬」→ +2分
- 当天「忌」列表包含上述关键词 → 直接标为「不宜」

### 维度二：黄道/黑道日（十二建星）
- 黄道日（青龙、天德、玉堂、司命、明堂、金匮）→ +1分
- 黑道日（天刑、朱雀、白虎、天牢、玄武、勾陈）→ -1分

### 维度三：个人冲煞（基于出生年份生肖）
- 当天地支与用户生肖形成「冲」关系 → 标注「冲 [生肖]，您不宜」，降为普通或不宜

### 最终评级

| 评级 | 显示颜色 | 条件 |
|------|----------|------|
| ⭐⭐ 大吉 | 深绿色 | 宜动土 + 黄道日 + 不冲本命 |
| ⭐ 吉 | 浅绿色 | 满足以上任意两条 |
| 普通 | 灰色 | 无明显宜忌 |
| ✗ 不宜 | 浅红色 | 忌动土，或冲本命 |

## 日期详情卡片内容

点击某天后弹出，用大白话显示：

- 公历日期 + 农历日期
- 黄道/黑道日名称
- 宜忌事项（摘选与动土相关的）
- 与用户生肖的关系（有冲则提示）
- 综合建议一句话：「✅ 适合修坟动土」或「❌ 今日不宜，建议另选」

## 适老化 UI 规范

| 元素 | 字体大小 |
|------|----------|
| 页面标题 | 28px |
| 日历日期数字 | 28px |
| 农历小字 | 16px |
| 正文/推荐文字 | 20px |
| 按钮文字 | 18px |
| 详情卡片文字 | 20px |

- 所有输入使用**下拉选择框**，不让老人手动输入数字
- 出生年份范围：1930–2010
- 查询年份范围：当前年份前后5年
- 按钮颜色醒目（深绿），大尺寸，易点击
- 大吉/不宜颜色一眼可辨，不依赖图标区分
